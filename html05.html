이미지 요소

*src: 이미지 소스의 경로 및 파일명

*alt 이미지 소스의 경로 및 파일명이 맞지 않거나 표시할 수 없는 경우의 대체 텍스트

오디오 요소

autoplay : 자동실행

loop : 반복실행

muted : 음소거

controls : 제어패널 보이기

비디오 요소

*src: 비디오 소스의 경로 및 파일명

브라우저에서 실행 가능한 비디오 형식 : mp4, ogv, webm

autoplay : 자동실행

loop : 반복실행

muted : 음소거

poster : 대기화면 이미지

controls : 제어패널 보이기

인프레임 요소

*src: 인프레임 문서의 경로 및 파일명

유튜브와 같은 인프레임 요소

옛날 방식인 embed, object 도 사용이 가능하나 웹 자원의 절약을 위해 잘 활용하지 않는다.
